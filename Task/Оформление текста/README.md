## Испытание: оформление статьи.

### Помните испытание про разметку в Глава 2 Тема: Разметка текста? Там вам нужно было разметить статью с помощью подходящих тегов.

### В этом испытании вам нужно оформить статью по испытание Разметка текста. Только, в этом испытании задача изменилась. Исходный HTML-код уже полностью готов, в нем ничего менять не надо. Вам нужно применить изученные в этой Главе CSS-свойства к соответствующим элементам.

## Итак, что нужно сделать:

— Для заголовка **задать размер шрифта 20px и горизонтальное выравнивание текста**(свойство `text-align`).

— Для блока `.highlight` **задать внутренний отступ 10px**, **белый цвет текста** (`#ffffff`) **и фон цвета** `#3f51b1`, а также поменять **насыщенность шрифта сделать ее толстый**.

— Для блока `.highdark` аналогично **задать внутренний отступ** 10px, **белый цвет текста** (`#ffffff`) и **фон цвета черный** , так же **поменять насыщенность шрифта на толстый**.

— В списке основные правила применить свойства `font-style`, `text-transform`, `text-decoration` с нужными значениями для соответствующих кусочков текста с классами `italic`, `uppercase`, `underline`.
