## ТЕХНИЧЕСКОЕ ЗАДАНИЕ:

1. Покраска модулей (Background Color).

Каждый блок должен иметь свой цвет идентификации:

— .block1: #0ea5ff (Синий)

— .block2: #ff7b00 (Оранжевый)

— .block5: #150050 (Темно-синий)

— .block6: #ff5e62 (Красный)

— .block7: #16222a (Темно-серый)

— .block8: #00f5a0 (Зеленый)

— .block9: #e100ff (Розовый)

2. Индикатор питания (Блок №1).

**В первом синем блоке должна быть иконка молнии.**

— Иконка: https://img.icons8.com/ios-filled/50/ffffff/lightning-bolt.png

— Требование: Она не должна «размножаться». Поставь её аккуратно: 15px от верха и 15px от левого края. Размер иконки сделай 40px.

3. Навигационные узлы (Блоки №4 и №6).

**Размести в них иконки планеты и ракеты.**

— Планета: https://img.icons8.com/ios-filled/100/ffffff/planet.png

— Ракета: https://img.icons8.com/ios-filled/100/ffffff/rocket.png

— Требование: Заказчик хочет видеть их строго по центру своих блоков. Размер — 60px.

4. Система «Стыковка» (Блоки №8 и №9).

Тут тебе нужно проявить смекалку. Мы хотим, чтобы один большой космонавт был разделен между двумя блоками. Если просто поставить картинку, она будет в каждом блоке своя и маленькая. Нам нужно сделать «пазл».

Твои действия:

— Единый размер: Установи для обоих блоков (.block8 и .block9) одинаковый размер фона background-size.

— Хитрость: Ширина одного блока 140px, их два, плюс зазор (gap) между ними 2px. Итого ширина картинки должна быть 282px. Высота остается 140px.

— Разные ракурсы: Представь, что картинка — это длинный плакат за двумя окнами.

В Блоке №8 (левое окно) тебе нужно прижать картинку к левому краю: background-position: left center;.

В Блоке №9 (правое окно) тебе нужно прижать ту же самую картинку к правому краю: background-position: right center;.

— Результат: Если ты всё сделал правильно, зазор в 2 пикселя идеально «проглотит» часть картинки, и космонавт будет выглядеть единым целым!
