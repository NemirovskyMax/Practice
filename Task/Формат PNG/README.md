# Задание: Формат PNG и JPEG

## Цель задания

В этом упражнении вы научитесь:

1. Подключать изображения форматов **PNG** и **JPEG** через тег `<img>`.
2. Использовать прозрачный фон для PNG-иконок.
3. Указывать путь к файлу, описание (`alt`) и ширину изображения (`width`).

## Что нужно сделать

Научится и понять принцип PNG картинок.
Сравнить PNG и JPEG картинки.

1. Сделайте картинку логотипа PNG
2. Сделайте иконку с задним фоном(фон указан в CSS)
3. Сравните JPEG и PNG картинки, посмотрите различия и поймите зачем нужны PNG картинки
