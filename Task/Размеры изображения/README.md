# Задание: Размеры изображений


## Цель задания

В этом упражнении вы научитесь:

1. Изменять размер изображения через `width`.
2. Изменять размер изображения через `height`.
3. Понимать, что происходит при нарушении пропорций.
4. Подбирать аккуратный размер изображения.
5. Сравнивать разные размеры одной и той же картинки.


## Что нужно сделать

В этом задании используется одно изображение `cat.jpg`.

Ваша задача — поэкспериментировать с размерами и понять, как работает изменение ширины и высоты.


## Работа с TODO

В HTML-файле есть комментарии `TODO`.

Ваша задача:

- Вставить правильный путь к изображению.
- Заполнить `alt`.
- Указать размеры в соответствии с подсказками.


### 1. Задаем только ширину

- Укажите ширину от 1 до 300px.
- Высоту не указывайте.
- Посмотрите, как браузер автоматически сохраняет пропорции.


### 2. Задаем только высоту

- Укажите высоту от 1 до 200px.
- Ширину не указывайте.
- Обратите внимание, как автоматически подстраивается ширина.


### 3. Нарушаем пропорции

Чтобы нарушить пропорции, нужно:

- Поставить большую ширину (например, 300px).
- Поставить маленькую высоту (например, 100px).

Посмотрите, как изображение "растягивается".


### 4. Подбираем аккуратный размер

- Укажите только ширину примерно 150–200px.
- Добейтесь аккуратного и пропорционального отображения.


### 5. Сравнение размеров

- Первой картинке задайте небольшую ширину (примерно 150–200px).
- Второй картинке задайте большую ширину (примерно 400–500px).
- Сравните визуально разницу.
