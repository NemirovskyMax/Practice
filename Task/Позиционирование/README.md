## Header (шапка).

— .fixed-header — наша шапка должна всегда оставаться наверху, даже когда мы скроллим страницу.

— position: fixed ➡️ фиксирует элемент относительно окна браузера.

— top: 0; left: 0 ➡️ ставим её в верхний левый угол.

— z-index: 1000 ➡️ гарантирует, что шапка будет выше всех остальных блоков.

     Подсказка: если убрать z-index, шапка может оказаться за другими блоками при перекрытии.

## Контейнер (container).

— .container-relative — это основной блок, внутри которого мы размещаем наши маленькие блоки (Box 1–5).

— position: relative ➡️ превращает контейнер в “точку отсчёта” для всех блоков с absolute.

### Подсказка: без этого внутренние блоки с absolute будут ориентироваться на всю страницу, а не на контейнер.

## Абсолютные блоки (Box 1–5).

— .box-position — общий селектор для всех наших блоков с абсолютным позиционированием:

     — position: absolute ➡️ позволяет ставить блок в конкретное место внутри контейнера.

### Подсказка: без absolute блоки просто будут “стоять” в потоке документа, как обычные div.

— Box1 — левый нижний угол:

     — bottom: 20px; left: 20px ➡️ блок не прижимается к краю, а немного отступает.

### Комментарий: выглядит аккуратно и не сливается с краем контейнера.

— Box2 — примерно посередине контейнера:

     — top: 130px; left: 300px; z-index: 2.

### Комментарий: z-index выше default, чтобы он оказался поверх Box1, если они перекрываются.

— Box3 — рядом с Box2, немного выше по слою:

     — top: 130px; left: 350px; z-index: 3.

### Комментарий: используем больший z-index, чтобы этот блок был выше Box2 и выделялся.

— Box4 — правый нижний угол:

     — bottom: 20px; right: 20px.

### Комментарий: симметрично Box1, но справа.

— Box5 — правый верхний угол:

     — top: 10px; right: 10px.

### Комментарий: закрепляем блок в верхнем правом углу, красиво смотрится вместе с header.

## Relative Box.

— .relative-box — блок, который сдвинут относительно своего обычного места, но остается в потоке страницы:

     — position: relative.

     — top: -70px; left: 300px.

### Комментарий: благодаря relative блок “поднимается” вверх на 70px, но не перекрывает соседние блоки полностью. Отлично подходит для небольших корректировок расположения.

## Fixed Button.

— .fixed-btn — кнопка, которая всегда видна в правом нижнем углу:

     — position: fixed.

     — bottom: 20px; right: 20px.

Комментарий: не зависит от прокрутки. Удобно для кнопок “Наверх” или “Заказать”, которые должны быть всегда на виду.
