## Задание: Галерея изображений-ссылок


### Цель задания

#### В этом упражнении вы научитесь:

1. Делать несколько изображений кликабельными.

2. Правильно заполнять атрибут `href`.

3. Использовать `target="_blank"` для открытия в новой вкладке.

4. Проверять корректность атрибута `width`.

5. Заполнять осмысленный `alt`.


## Что нужно сделать

В HTML-файле есть 7 изображений.

#### Каждое изображение должно:

- Быть обёрнуто в тег `<a>`.

- Вести на свою ссылку.

- Открываться в новой вкладке.

- Иметь корректный `alt`.

- Иметь указанную ширину.
