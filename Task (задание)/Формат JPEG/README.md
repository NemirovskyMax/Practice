# Задание: Формат JPEG


## Цель задания

В этом упражнении вы научитесь:

1. Подключать изображения формата JPEG через тег `<img>`.
2. Указывать путь к файлу, описание (`alt`) и ширину изображения (`width`).


## Что нужно сделать

В `<img>`:
- Вставить в `src` blog.jpg
- Дать ширину `width` примерно 400-500px

## Структура проекта

Проверьте, что у вас есть папка с изображением:
```
Practice
   └── Practice
        └── images
             └── blog.jpg
```
Если изображение не отображается — проверьте путь к файлу.
