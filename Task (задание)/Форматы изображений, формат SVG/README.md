# Задание: Форматы изображений (PNG и SVG)


## Цель задания

В этом задании вы научитесь:

1. Подключать изображения через тег `<img>`.
2. Работать с форматами PNG и SVG.
3. Указывать правильный путь к файлу.
4. Задавать размер изображения через атрибут `width`.


## Что нужно сделать в каждом `<img>`

Цифра это нумировка `<img>`:
1. Вставить после .svg и дать размер 120px
2. Поменять logo.png на svg
3. Вы должны вставить logo.png и дать размер 300px
4. Вы должны вставить logo.svg и дать размер 300px
5. В 3,4 можете играться с размерами, смотреть как меняется изображения
