# Задание: Якорные ссылки и длинная страница


## Цель
Научиться:


- Делать страницу с несколькими разделами
- Создавать **якорные ссылки**, чтобы быстро переходить к разделам
- Делать возврат к началу страницы

CSS уже подключён и отвечает за стили (цвет текста, шрифты, отступы). Ваша задача - **структура и логика HTML**.


## Подсказки по HTML


### Создаём навигацию

- Используйте `<ul>` + `<li>` для списка разделов.
- Внутри `<li>` добавляем `<a href="#id">Название раздела</a>`  

Например:

```html
<ul>
  <li><a href="#section1">Раздел 1</a></li>
  <li><a href="#section2">Раздел 2</a></li>
</ul>
```

## Разделы страницы

- Каждый раздел оформляем через `<h2>` с уникальным атрибутом `id`, совпадающим с `href` в ссылках.

Например:

```html
<h2 id="section1">Раздел 1</h2>
<p>Текст раздела…</p>
```

## Ссылки "Наверх"

- Чтобы вернуться к началу страницы, создайте `<a href="#body">Наверх</a>`.

- Для этого добавьте `id="body"` к тегу `<body>`:
```html
<body id="body">
```

## Общие советы

- Атрибут `id` должен быть уникальным для каждого раздела.

- Старайтесь, чтобы текст был длиннее, тогда появится полоса прокрутки.

- Используйте `<hr>` для визуального разделения разделов.

- Не трогайте CSS, но наблюдайте за стилями — они помогут понять расположение текста и блоков.
