# Задание: Атрибут `alt`


## Цель задания

### В этом упражнении вы научитесь:

1. Понимать, зачем нужен атрибут `alt`.

2. Отличать декоративные изображения от смысловых.

3. Правильно заполнять `alt` для контентных картинок.

4. Использовать пустой `alt=""` для декоративных изображений.

5. Делать разметку более доступной и правильной.


## Что нужно сделать

### В HTML-файле уже добавлены три изображения:

- Фоновая картинка

- Кот

- Логотип

### Ваша задача — правильно заполнить атрибут `alt` в каждом случае.

## Работа с TODO

В HTML-файле есть комментарии TODO.

### Ваша задача:

- Определить, несёт ли изображение смысл.

- Если изображение декоративное — указать пустой `alt=""`.

- Если изображение контентное — описать его кратко и понятно.

# Подсказки
## 1. Фоновая картинка

### Подумайте:

- Несёт ли она смысл?

- Изменится ли понимание страницы, если её убрать?

### Если картинка используется просто как фон или украшение —
укажите:

`alt=""`

## 2. Кот

Это содержательная картинка.

### В `alt` нужно:

- Кратко описать, что изображено.

- Не писать слишком длинное описание.

- Не использовать фразы типа "картинка с…".

### Пример хорошего варианта:
```
Кот в коробке
```

## 3. Логотип

Если логотип является частью контента, его нужно описать.

### Правило:

- Указать название бренда.

- Не писать слово "логотип", если это очевидно по контексту (но в учебных заданиях можно).

### Пример:
```
Mobile Legend Bang Bang
```

## Важно помнить

Если убрать изображение и смысл страницы не меняется —
это декоративная картинка → `alt=""`.

Если изображение несёт информацию —
нужно краткое и понятное описание.
