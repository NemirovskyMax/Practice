# Задание: Формат GIF


## Цель задания

В этом упражнении вы научитесь:

1. Подключать анимированное изображение формата GIF.
2. Правильно заполнять атрибуты `src`, `alt` и `width`.
3. Работать с фоном страницы через CSS.


## Что нужно сделать

Научиться и понять принцип работы GIF-картинок.  
Посмотреть, чем GIF отличается от обычных PNG и JPEG.

1. Вставьте GIF-картинку в HTML.
2. Напишите корректный `alt`.
3. Укажите ширину изображения.
4. Задайте цвет фона страницы через CSS.


## Работа с TODO и !

В HTML-файле есть комментарий `TODO`.

Ваша задача:

- Заменить пропуски `______`.
- Указать правильное имя файла GIF.
- Заполнить `alt`.
- Установить подходящую ширину.

## Немного про CSS

чтобы увидеть прозрычный ли фон у GIF, даем всей страницы `body` цвет через команду `background-color:(цвет в нашем случае #999);`
