Приветствую, здесь будет гайд по репозиторию. Как его легко клонировать через Git.

## Шаг первый скачайте Git.

- Вы заходите на сайт [Git](https://git-scm.com/install/)
- Скачиваете на свое устройсто, виндовс, мак и тд
- В установщике ничего не меняем просто нажимаем next
- Потом заходите в папку Git куда вы скачали
- Ищите git-bash.exe открываете и вот у вас Git

## Шаг второй настраиваем Git.

- git config --global user.name "ИмяВаше"
- git config --global user.email "имейл@gmail.com"
- Это нужно, чтобы Git знал, кто делает изменения
- Действует для всех репозиториев на этом компьютере

Проверяем настроилось ли:

- Пришем git config --list
- Почти в самом низу ищем user.name и user.email
- Должно быть ваши имена и имейлы
- Если есть вы все правильно делаете

## Шаг третий настрой VS Code как редактор

Если хочешь, чтобы Git открывал VS Code при необходимости редактировать коммиты:

- git config --global core.editor "code --wait"
